### 交易接口（以下报文全部说明的是报文体body的参数，且均为正常流程，具体数据结构暂时尚未给出）
#### 开通汇付账户：
* 应用场景：用户跳转到汇付页面开通汇付账户
* cmdId： **OpenPnrAccount**
* 请求方式：GET
* request：无

* response：跳转到汇付开户页面

#### 获取用户汇付账户信息：
* 应用场景：获取用户汇付账户信息
* cmdId： **GetUserAccount**
* request：无
* response：

|参数名|参数描述|必要|参数校验|备注|
|-|-|-|-|-|
|account|用户汇付账户|Y| |包含汇付用户名，状态|

account详细字段：
```json
    account:{
       name:[姓名], 
       usrId:[汇付账户用户名],
       usrStat:[汇付账户状态],
       createTime:[开通时间]
    }
```

#### 充值：
* 应用场景：用户充值
* cmdId： **Deposit**
* 请求方式：GET
* request：

|参数名|参数描述|必要|参数校验|备注|
|-|-|-|-|-|
|amount|充值金额|Y| |充值金额单位元|
|gateBusiId|充值网关|Y| |QP-快捷卡充值、B2C-网银充值|
|openBankId|开户银行代号|Y| |代号见附件银行代号|
|actionFlag|差额充值标志位|N| |差额充值标志位（0,1）|

* response：后台处理后跳转到汇付充值页面

* 应用场景：获取用户投资记录列表

* response：


|参数名|参数描述|必要|参数校验|备注|

userInvestList详细字段：
```json
    investList:[
        {

            years:[标的投资期限年],
            months:[标的投资期限月],
            days:[标的投资期限天],
            repayMethod:[标的还款方式],
            amount:[投资金额],
            investTime:[投资时间],
            status:[投资状态],
            receivable:[待收本息（针对还款中的投资）],
            interest:[利息收益（针对已经还款的投资）],
            repayedTime:[还款时间（针对已经还款的投资）]
        },
        {


        }
    ]
```
